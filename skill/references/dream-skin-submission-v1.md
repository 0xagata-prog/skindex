# Dream Skin submission v1

SkinDex is the catalog and review layer; Codex Dream Skin is the complete-skin runtime source.

For every Dream Skin submission, collect and disclose:

- `engine`: `dream-skin`
- `capabilities`: only features actually implemented (`background`, `palette`, `icons`, `layout`, `motion`, `companion`, `custom-ui`)
- supported platform
- whether it was verified in a real Codex window
- HTTPS source URL when the project already has a repository
- a review thumbnail no larger than 700 KB

Do not upload or execute the local project, installer, injected JavaScript, credentials, browsing data, full-resolution private assets, or Codex application files. A reviewer must separately inspect licensing, the upstream runtime boundary, screenshots, and claimed capabilities before catalog publication.

The first “yes” after creation means the user is interested in submitting. A second explicit confirmation is required after showing the exact metadata and thumbnail that will be uploaded.
